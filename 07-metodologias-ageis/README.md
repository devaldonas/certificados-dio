## 🎯 Principais Aprendizados

### Métodos Ágeis
- **Scrum:** Papéis (Product Owner, Scrum Master, Time), cerimônias (Sprint Planning, Daily, Review, Retrospective)
- **Kanban:** Quadro visual, WIP (Work in Progress), fluxo contínuo
- **XP (Extreme Programming):** Programação em par, TDD, integração contínua

### Ciclo de Desenvolvimento
- Levantamento de requisitos
- Planejamento e estimativas
- Desenvolvimento iterativo
- Testes e garantia de qualidade
- Entrega e deploy
- Manutenção e feedback

## 💼 Aplicação no dia a dia

Estas metodologias são aplicadas diariamente no desenvolvimento dos meus projetos, garantindo entregas contínuas e organizadas, com uso de:
- GitHub Projects para gerenciamento de tarefas
- Issues e Pull Requests para controle de versões
- Milestones para planejamento de sprints
