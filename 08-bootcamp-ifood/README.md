**Instituição:** DIO.me em parceria com iFood

**Carga Horária:** Programa completo com múltiplos módulos

## 📋 Sobre o Bootcamp

O bootcamp **Potência Tech iFood - Programação do Zero** foi um programa intensivo de capacitação em desenvolvimento de software, com foco em fundamentos da programação e introdução ao desenvolvimento web.

## 🎯 Principais Aprendizados

### Fundamentos da Programação
- Lógica de programação
- Algoritmos e estruturas de dados
- Variáveis, operadores e estruturas de controle
- Funções e modularização

### Desenvolvimento Web
- Introdução ao HTML, CSS e JavaScript
- Versionamento com Git/GitHub
- Conceitos de front-end e back-end

### Soft Skills
- Resolução de problemas
- Pensamento computacional
- Trabalho em equipe e colaboração (Git/GitHub)

## 🚀 Como este bootcamp impactou minha trajetória

Este bootcamp foi o **ponto de partida** da minha transição de carreira para desenvolvimento de software. A partir dele:
- Desenvolvi meus primeiros projetos práticos
- Criei meu portfólio no GitHub
- Conquistei certificações complementares (Python, Banco de Dados, etc.)
- Comecei a cursar **Engenharia de Software**
